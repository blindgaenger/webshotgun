# WebShotGun

Provides a Speed Dial like interface for webpages. Webshots are generated locally via webkit2png.

## Requirements

* ruby
* bundler (http://gembundler.com/)
* Mac OS X
* Python
* webkit2png (http://www.paulhammond.org/webkit2png/)

## Usage

* bundle install
* rake db:migrate
* rake db:seed
* rake jobs:work
* ./script/server
* http://localhost:3000/
